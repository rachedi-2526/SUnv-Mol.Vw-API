Cartoon, Stick, Line, Sphere, and Surface rendering options

Dynamic Color Schemes

Secondary Structure, Element, Chain, and pLDDT coloring

Chain Management

Multi-chain support with individual chain toggling

Confidence Visualization

pLDDT confidence scores for AlphaFold structures

📱

Responsive Design

Optimized for desktop and mobile devices

🏗️

Large Structure Support

🔧 Integration Examples

Direct URL Usage

Use the API directly with URL parameters:

SUnv-Mol_Vw_api.html?entId=1CRN
SUnv-Mol_Vw_api.html?entId=AF-Q9I4X8-F1

Embed in Web Page

Embed the viewer in your website:

<iframe src="SUnv-Mol_Vw_api.html?entId=1CRN" width="800" height="600"></iframe>

JavaScript Integration

Load structures programmatically:

function loadStructure(id) {
  window.open('SUnv-Mol_Vw_api.html?entId=' + id, '_blank');
}
